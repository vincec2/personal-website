Cloud Productivity Dashboard

Cloud Productivity Dashboard (CPD) is an windows software created with React and ElectronJs. It saves projects and their tasks inside of a .json file in the user's local computer's AppData/Roaming. Features include multiple settings to customize the dashboard, a countdown/pomodoro. Check link in the README.md in the repository to check out the software!
